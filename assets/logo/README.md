# Logo PREV Hub

Placez votre logo PREV Hub dans ce dossier :

1. **prevhub_logo.png** - Logo complet (1024x1024px recommandé)
   - Format: PNG avec transparence
   - Taille minimale: 512x512px
   - Taille recommandée: 1024x1024px
   - Le logo orange PREV Hub sur fond transparent ou noir

2. **prevhub_logo_foreground.png** (optionnel) - Version pour icône adaptative
   - Format: PNG avec transparence
   - Taille: 1024x1024px
   - Le logo seul sans fond (pour l'icône adaptative Android)

## Instructions

1. Ajoutez votre fichier `prevhub_logo.png` dans ce dossier
2. (Optionnel) Ajoutez `prevhub_logo_foreground.png` pour l'icône adaptative
3. Exécutez: `fvm flutter pub get`
4. Exécutez: `fvm flutter pub run flutter_launcher_icons`

Les icônes seront générées automatiquement dans tous les dossiers Android nécessaires.
